## [0.1.0] - 2024-XX-XX

### Added

- Initial release
- `MapContainer` component for creating maps
- `TileLayer` component for adding tile layers
- `Marker` component for adding markers
- `Popup` component for marker popups
- `GeoJSON` component for rendering GeoJSON data
- `useMap` hook for accessing map instance
- `useMapEvents` hook for subscribing to map events
- `MapProvider` context for map state management
- Full TypeScript support
- Storybook documentation
- Comprehensive test suite
