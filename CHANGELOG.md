# Changelog

## 0.2.0

### Minor Changes

- Initial release of motion-map-components - Modern React components for interactive maps with Leaflet.

  Features:
  - MapContainer component with TypeScript support
  - Marker component with customizable icons
  - Popup component for interactive content
  - Full Leaflet integration
  - TypeScript definitions
  - CSS styling included

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [0.1.0] - 2024-XX-XX

### Added

- Initial release
- `MapContainer` component for creating maps
- `TileLayer` component for adding tile layers
- `Marker` component for adding markers
- `Popup` component for marker popups
- `GeoJSON` component for rendering GeoJSON data
- `useMap` hook for accessing map instance
- `useMapEvents` hook for subscribing to map events
- `MapProvider` context for map state management
- Full TypeScript support
- Storybook documentation
- Comprehensive test suite
